# LinkAndroid

## 软件介绍

LinkAndroid 是一个安卓手机连接管理器，可以连接多个手机，一键投屏，手机操作。

![](https://ms-assets.modstart.com/data/image/2024/09/17/29242_pefp_8591.png)

![](https://ms-assets.modstart.com/data/image/2024/09/17/29294_q61z_7460.png)

## 技术栈

- `Electron`
- `Vue3`
- `scrcpy`
- `typescript`

## 功能特性

- 多手机控制
- 一键投屏
- 手机操作
- 国际化，支持简体中文、英语

## 二次开发

```
npm install
npm run dev
```
